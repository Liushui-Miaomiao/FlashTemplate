# Flash 模板 (Flash Template)

## 概述

这是一个模板集合，包含了历代 Adobe Flash/Animate（包括 Macromedia Flash 8 ）软件中，用于新建 ActionScript 2 / ActionScript 3 / Adobe AIR 的空白 .fla 模板。

除了上述模板，还附带了每一代软件包内自带的原生 播放器（Flash Player）/ 运行时（AIR Runtime）。

为了方便做绿色软件换壳，预编译了一套带有 32 * 32 像素图标的 AIR APP。可根据自身需要通过替换相关资源实现对部分 Flash 技术栈实现的软件进行绿化。
